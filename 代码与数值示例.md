## 一、先给出【要对照的代码结构编号】

为了能在表格里准确写清楚“执行了哪部分代码”，先把核心代码**编号**：

```python
① for i in range(len(nums)):

②     if q and q[0] <= i - k:
③         q.popleft()

④     while q and nums[q[-1]] < nums[i]:
⑤         q.pop()

⑥     q.append(i)

⑦     if i >= k - 1:
⑧         res.append(nums[q[0]])
```

后面表格中的“执行代码”一列，**只引用这些编号**。
